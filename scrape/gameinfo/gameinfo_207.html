Default FOV:
Field of View: 55 | Vertical Degrees
 
Edit the following key as DECIMAL in the registry to adjust FOV: HKEY_CURRENT_USER\SOFTWARE\CampoSanto\Firewatch\fovAdjust_h2041137991. Enter the value you want to increase the FOV by multiplied by 100. So if you want to increase it to 80, enter 2500 ((80-55)*100). Note that the value is vertical FOV, which can be calculated from horizontal FOV in the calculator.
