Notes:
The actual sensitivity value in the config file is "MouseSensitivity", but if you try to enter a value here, it will be overwritten by the game. The trick is to adjust the scale of the in-game sensitivity slider. So use the value you get from the calculator on both "MaxSensitivityMultiplier=" and "MinSensitivityMultiplier=" under the [TdGame.TdPlayerInputConsole] section (important!). This will make the whole slider be the value you want, so you do not need to do anything else, just start the game!
