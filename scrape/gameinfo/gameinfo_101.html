Notes:
The console needs to be enabled before you can access it. Edit the client.ini file in the install folder or in the live folder depending on your installation. Add the following:
application {
enable_console=true
}
If you already have an empty "application" section in the file, just add enable_console=true inside it. Set both FOV and sensitivity from the console.
