Notes:
While it is possible to set the sensitivity in the menus, the preferred way is using the console. The reason for this is that the value you set in the menus are not the exact value applied. For instance can 0.24 be applied as something like 0.2361809. After you have changed the settings, you need to quit and re-enter the world for them to take effect.
