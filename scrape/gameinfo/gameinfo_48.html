Notes:
Since Rage do not allow you to adjust the mouse sensitivity other than with the slider in the menu (none of the m_sensitivity commands etc work), here's what you do to get a precise value: Set the Mouse Sensitivity to the lowest in-game. Click a few more times after you don't see the slider any more to make sure it's all the way down. Now use the calculator to calculate the m_yaw value based on distance or another game. You need to create an empty file on the base folder named "rageconfig.cfg". To enable console commands, put "seta com_AllowConsole 1" as the first line. For the next lines enter the same calculated for m_yaw AND m_pitch, default is "m_yaw 0.022" and "m_pitch 0.022".
