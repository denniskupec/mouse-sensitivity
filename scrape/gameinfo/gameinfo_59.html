Notes:
Each click with the arrow keys adjust the sensitivity by 5%. So if you want a sensitivity of e.g. 10%, move the sensitivity slider all the way down, and click right arrow twice. If you want to be even more granular and set it to for example 7%, move the slider back a few pixels with the mouse. Since the lowest sensitivity in Warframe is quite high, the recommended way of getting the distance you want is to set the sensitivity to 0% and adjust the DPI until you get it right.

Affected by:
FOV: Yes | WPS: No | Resolution: No
Default FOV:
75 | Horizontal Degrees (4:3 Base)
 
Adjust FOV with the in-game slider under Options->Display. The slider ranges from 60 to 90 degrees, 75 is the default. Use the arrow keys do adjust in steps of 1.5 degrees (check the notes if the arrow keys do not work). This means that if you for instance want an FOV of 84, this is 16 clicks from 0 (16*1.5+60).
