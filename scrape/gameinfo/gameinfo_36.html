Sensitivity values below 2 result in a lot of count loss, to the point where they do not get registered at all. For this reason, 2 is set as a minimum. There's also a small amount of unavoidable acceleration. The in-game settings allow you to set up to four decimals, so it might not be necessary to edit the file if this is enough for you.
