Notes:
Find the line in the config file that starts with StringData=(Tag="ControlsMouseSensitivity", and edit the DefaultValueIndex value in this line to adjust the mouse sensitivity. Ignore the CurrentValue setting, as changing this will not affect the sensitivity. The calculated value is 10x that of the sensitivity in-game, if you want to set it there, just devide the sensitivity you get by 10.
