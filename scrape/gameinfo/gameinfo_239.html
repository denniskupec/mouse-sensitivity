Notes:
Set mouse acceleration to 0 by either launching the game with the -console option and setting "mouse_acceleration 0" in the console, or by making a file in the game folder containing the option and launch it with the game with the -exec filename option.
Even with acceleration set to 0 there is quite a lot of it present, especially when moving the mouse quickly. These calculations are for unaccelerated movement.
Sensitivity 9 and 10 are the same, so 9 is set as maximum.
