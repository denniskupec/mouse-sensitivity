Notes:
Adjust the sensitivity with arrowkeys in-game under Options -> Controls -> Mouse Sensitivity. Each click left or right is 1 step, and the sensitivity bar has 60 steps, ranging from 1-60. So to set a sensitivity of e.g. 9, move the slider all the way to the left, and press the right arrowkey 9 times.

Affected by:
FOV: No | WPS: No | Resolution: No
Default FOV:
FOV: 52 | Vertical Degrees
 
Configure FOV by adding/editing the ExtraGameFov(XX.XX) variable in the video.scr file located in %USERPROFILE%\Documents\DyingLight\out\settings. The value you set is added to the default of 52, so if you want a vertical FOV of 70, you set ExtraGameFov(18.00).
