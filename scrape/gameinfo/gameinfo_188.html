Notes:
Current version (1.109) sufferes from quite a lot of packet drop and acceleration. To avoid packet loss, use a sensitivity value above 1. And to avoid acceleration, use a sensitivity without digits (so either 1, 2 or 3). This obviously limits your options, but also eliminates all acceleration.
Note that the sensitivity value will not be in the file until you have changed it once in-game.
