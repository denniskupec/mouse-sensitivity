Notes:
Use this to calculate the relative sensitivity for all zoom values. For this to work, they must all be set to the same. The FOV for 4.00X is used in the calculations, but the relative movement will be the same for all aims.
Use this calculation if "ADS Field of View" is off. It's important to enter your configured FOV.
This calculation require the following settings:
GstInput.MouseRawInput 1
GstInput.SoldierZoomSensitivity#x## ALL THE SAME
GstInput.UniformSoldierAiming 1
GstRender.FieldOfViewScaleADS 0
The last setting is in the PROFSAVE_profile file.
Make sure you enter the correct values into the fields, as the fields can change depening on the aim you select.
Engine:
Frostbite 3
Affected by:
FOV: Yes | WPS: No | Resolution: Yes
Default FOV:
GstRender.FieldOfViewVertical 55 | Vertical Degrees
 
FOV can be set in-game between 50 and 100, and in the config file with the parameter "GstRender.FieldOfViewVertical".
