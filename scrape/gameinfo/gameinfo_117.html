Notes:
Manually setting the sensitivity in the config file will be overwritten when you start the game, so you need to set the sensitivity via the console when the game is running. The setting will not be saved, so you need to enter it everytime the game launches (for now anyway). Due to the nature of the movement in this game, sensitivity may feel different than other games. Make sure mouse smoothing is turned off.
