Notes:
This game scales differently with FOV than other CoD games.
Use the arrow keys to adjust the sensitivity slider. Move it all the way to the left first, this is equal to 1. One click on the right arrow key will increase it to 2 and so on. Set Mouse Vertical Sensitivity to the same if you want them to match.
Use the HEX mode for accurate sensitivity.
