Notes:
Aimed sensitivity inherits the settings from hipfire sensitivity, there is no specific setting for it. The correct mouse sensitivity setting is under [Engine.PlayerInput].
IMPORTANT: Make sure this setting is set to 1:
LookSensitivityMultiplier=1.000000
If you have adjusted the sensitivity in-game, this value will be different. Disable acceleration by setting
bEnableMouseSmoothing=false
in the config file.
