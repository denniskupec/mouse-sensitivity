Notes:
Adjust the sensitivity with arrowkeys in-game under Options -> Controls -> Mouse Sensitivity. Each click left or right is 1 step, and the sensitivity bar has 100 steps, ranging from 0-100. So to set a sensitivity of e.g. 9, move the slider all the way to the left, and press the right arrowkey 9 times.
Editing the GstInput.MouseSensitivity value in the config file %USERPROFILE%\Documents\Plants vs Zombies GW2\settings\PROF_SAVE_profile does not work, as it will be overwritten when the game is started. Instead, adjust the sensitivity in-game, and check the value in the config file. Devide the calculated value by 100 to get the value in the config file. E.g. 25 calculated = 0.25 in the config file.
